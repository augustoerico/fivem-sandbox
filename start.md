# Start

1. Player create a character: 
    - first_name
    - last_name
    - gender
    - [optional] customize:
        - underwear (male: boxer, female: panties and sports top)
        - basic clothes (tdb)

2. Player earn $2500 and a light
    - car (not motorcycle): mandatory
    - clothes
    - gear and melee weapon ("hunting"gear)

3. Spawn point: "albergue"
